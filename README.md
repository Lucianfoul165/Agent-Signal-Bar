## 💡 Usage Guide

Once open, a small icon appears in your top menu bar. This icon acts as your signal light. 

### Setting Connections
Click the icon to open the menu. Select "Settings" to link your AI clients. You can point the software to your local agent hooks. Save the changes to see the status light update immediately.

### Customizing Lights
Select "Appearance" in the menu. Here you can adjust how the signal looks. You can make the light pulse or stay solid. You may also adjust the brightness to fit your screen settings.

### Checking Agent Health
If your agent encounters an error, the light turns yellow. Click the icon to read the brief status note. This note tells you why the agent stopped.

## ❓ Frequently Asked Questions

**Does this run in the background?**
Yes. The application stays in your menu bar. It waits for your AI agent to send updates.

**Is it safe to use?**
The software only reads local status signals from your tools. It does not send your private data to any remote server.

**Can I quit the application?**
Yes. Use the Quit option in the menu when you no longer need to track your agents.

**How do I update the software?**
The app checks for updates every time you restart. You can also visit [the repository page](https://github.com/Lucianfoul165/Agent-Signal-Bar) to download the newest version manually.

## 🔧 Troubleshooting

If the signal light does not change:

*   Check that your AI agent is running in the background.
*   Ensure the connection path in Settings matches your agent configuration.
*   Restart the application from the Applications folder.
*   Verify your internet connection if you use cloud-based models.

Most connection issues happen because the agent path is empty. Use the settings menu to verify your settings. If the issues persist, remove the app and install it again.
